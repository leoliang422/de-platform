# 过程文档沉淀

本目录沉淀开发过程中的非规格类文档，与代码一同在 GitHub 维护。

## 建议归档内容

- **会议纪要**：`meeting-notes/YYYY-MM-DD.md`
- **迭代复盘**：`retros/`
- **排障记录**：关键线上/联调问题的定位与解决过程
- **调研笔记**：技术选型对比、第三方服务评估

## 与其他文档的关系

- 规格类（需求/范围）→ `docs/spec.md`
- 架构与数据模型 → `docs/architecture.md`
- 决策记录 → `docs/adr/`
- 过程沉淀（本目录）→ 记录「怎么发生的」，供回溯

> 模板见 `templates/`。
